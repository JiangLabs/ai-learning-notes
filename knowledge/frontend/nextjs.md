# Next.js

## 主题说明
学习用 Next.js 构建和部署现代 Web 应用，尤其是带服务端能力的 AI 产品原型。

## 核心概念
App Router、Server / Client Components、Route Handlers、缓存、流式渲染与部署。

## 我需要理解的问题
- [ ] Server 和 Client Component 如何划分？
- [ ] AI 流式输出应放在哪一层？
- [ ] 从原型到生产还缺哪些能力？

## 和产品工作的关系
Next.js 能把原型、前端和轻量后端放在同一项目，适合快速验证 AI 体验。
