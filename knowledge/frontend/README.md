# Frontend

## 文件用途

记录构建 AI 产品所需的前端基础与实践，连接交互原型、流式 AI 体验和可维护工程。

## 核心问题

- 怎样把 AI 的流式、工具调用和不确定状态做成清晰界面？
- 从原型到可发布产品需要补齐哪些前端能力？
- 如何用类型和组件降低复杂交互的维护成本？

## 学习 / 记录模板

每个主题记录核心概念、最小实验、常见错误、和 AI 产品的关系，以及下一次要实现的功能。
