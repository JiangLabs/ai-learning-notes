# Coding Agent

## 文件用途

研究 Coding Agent 如何理解代码库、规划修改、调用开发工具，并在人类监督下完成工程任务。

## 核心问题

- Coding Agent 如何建立代码库上下文并安全修改文件？
- 不同产品怎样处理计划、权限、验证和用户接管？
- 哪些 Coding Agent 设计可以迁移到其他专业 Agent？
