# Cursor

## 主题说明
研究 AI 如何进入编辑器，关注补全、对话、代码库理解和 Agent 模式。

## 核心概念
Inline Completion、Codebase Indexing、Agent、Rules、上下文引用与修改审查。

## 我需要理解的问题
- [ ] 补全、问答和 Agent 如何分工？
- [ ] 上下文选择怎样减少沟通成本？
- [ ] 用户怎样快速审查大范围修改？

## 和产品工作的关系
Cursor 是“AI 嵌入专业工具”的代表，可用于研究何时建议、何时等待和如何保留控制权。
