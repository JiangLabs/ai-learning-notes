# Plan Mode

## 主题说明
研究 Agent 如何在执行复杂任务前探索、规划和确认，以及怎样动态更新计划。

## 核心概念
只读探索、任务拆分、依赖、计划确认、权限边界与偏差处理。

## 我需要理解的问题
- [ ] 什么任务值得进入 Plan Mode？
- [ ] 好计划需要细到什么程度？
- [ ] 计划变化时如何解释原因？

## 和产品工作的关系
Plan Mode 是高风险操作中的信任机制，让用户从结果验收者变成过程合作者。
