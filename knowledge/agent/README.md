# Agent

## 文件用途

记录 AI Agent 的核心机制，理解一个 Agent 怎样观察、规划、使用工具、维护上下文并完成长任务。

## 核心问题

- Agent 如何从用户目标持续推进到完成状态？
- Tool、Skill、MCP、Context 和 Memory 怎样协作？
- 如何在自主性、可靠性、成本和用户控制之间取舍？
