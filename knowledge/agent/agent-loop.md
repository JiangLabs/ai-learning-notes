# Agent Loop

## 主题说明
理解 Agent 如何在模型推理、工具执行和环境反馈之间循环，直到完成目标或需要用户介入。

## 核心概念
观察、决策、行动、工具结果回传、停止条件与失败恢复。

## 我需要理解的问题
- [ ] 谁负责驱动循环，模型还是宿主程序？
- [ ] 如何判断任务已经完成？
- [ ] 工具失败或超时时怎样恢复？

## 和产品工作的关系
Agent Loop 决定用户看到的等待、进度、确认和失败体验。产品要把关键状态翻译成可理解的反馈。
