# Roadmap

## 文件用途

维护 AI Product Engineer 的长期能力路线、阶段重点和月度行动，帮助我在广泛主题中保持取舍。

## 核心问题

- 当前最需要补齐的能力短板是什么？
- 哪些知识必须通过项目而不是阅读掌握？
- 本月行动是否服务长期方向？

## 当前文件索引

- [ai-product-engineer-roadmap.md](ai-product-engineer-roadmap.md)：长期能力地图与阶段路线
- [monthly-plan.md](monthly-plan.md)：月度重点、产出与复盘模板

## 学习 / 记录模板

长期目标 → 当前阶段 → 能力差距 → 本月重点 → 可见产出 → 复盘调整。

## 后续更新方向

- 每月小幅调整，避免频繁重写长期目标。
- 用项目和工作成果，而不只用阅读数量判断进展。
