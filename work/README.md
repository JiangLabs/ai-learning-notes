# Work

## 文件用途

连接学习与真实工作，沉淀产品背景、关键问题、方案判断、验证结果和后续动作。

## 核心问题

- 当前产品最值得用 AI 解决的问题是什么？
- 技术机制怎样转化为用户价值和可控体验？
- 哪些判断已有证据，哪些仍只是待验证假设？
