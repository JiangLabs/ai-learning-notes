# Weekly Notes

## 文件用途

保存每周学习复盘，用短记录连接输入、真实理解、工作启发、项目进展和下周行动。

## 核心问题

- 本周真正理解了什么，而不只是看过什么？
- 哪个输入进入了工作或项目？
- 下周最值得继续的一件事是什么？

## 当前文件索引

- [2026-week-01.md](2026-week-01.md)：2026 年第 1 周复盘模板

## 学习 / 记录模板

学了什么 → 看了哪些项目 → 理解了哪些概念 → 工作启发 → 项目进展 → 下周计划 → 待提问问题。

## 后续更新方向

- 每周新增一篇，文件名沿用年份与周次。
- 每月回看四篇周记，将共性问题同步到 roadmap。
